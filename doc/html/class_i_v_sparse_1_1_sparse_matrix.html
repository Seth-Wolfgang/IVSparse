Detailed Description

template<typename T, typename indexT = uint64_t, uint8_t compressionLevel = 3, bool columnMajor = true>
class IVSparse::SparseMatrix< T, indexT, compressionLevel, columnMajor >

Template Parameters
TThe data type of the values in the matrix
indexTThe data type of the indices in the matrix
compressionLevelThe compression level used
columnMajorWhether the matrix is stored in column major format

A class to represent a sparse matrix compressed in the Compressed Sparse Fiber format (IVSparse).

IVSparse Sparse Matrix is a read-only matrix class optimized for sparse-dense computation in cases where values are highly redundant. For such cases, sparse fiber storage can reduce memory footprint by up to 50% compared to standard sparse compression. IVSparse also increases the ability to further compress index arrays within each run. This default templated version is for compression3 specifically. For compression level 1 and 2 there are template specializations.
